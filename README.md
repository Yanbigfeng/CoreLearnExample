## 静态文件和中间件的了解
对于资源文件例如图片可以做到任意目录访问。补充在core中资源文件的访问都是在wwwoot文件夹下的，其他目录默认是不可访问的。我们要做的就是可以在页面读取到其他目录的文件。这里就涉及到静态文件中间件的概念（UseStaticFiles）
第二个实现效果就是在做的开启任意目录浏览，该功能效果类似ftp模式可以通过路径查看到下面的资源列表。
